# 추천 유저 규칙 (User Rules)

아래 내용을 안티그래비티 설정 > User Rules에 복사해서 자신에 맞게 수정하세요.

---

```
언제나 한글로 대화하고 모든 문서와 설명도 한글로 작성한다.
아이콘은 "https://iconify.design/" 을 참고하거나, 이미지 생성 도구로 직접 제작한다.
반응형 웹 사이트 참고 : "https://mobbin.com/discover/sites/latest"
디자인 레퍼런스 참조 사이트 : "21st.dev" "dribbble.com" "goodly.website"
프로젝트를 설계하고, 페이지 디자인을 하기 전에 반드시 승인을 요청한다.
디자인에서 SVG 기능을 적극적으로 활용한다.
설치된 skills 와 mcp 를 활용하여 가장 유용한 개발 방법을 선택한다.
매 주 월요일에 새로운 스킬스를 검색하고 유용한 스킬스를 설치한다.
```

---

## 수정 가이드

| 항목 | 설명 | 예시 |
|------|------|------|
| 언어 | 기본 대화 언어 | 한글, English, 日本語 |
| 디자인 참조 | 디자인 영감을 얻을 사이트 | dribbble.com, behance.net |
| 승인 규칙 | 코드 작성 전 승인 필요 여부 | 프로젝트별로 조절 |
| 스킬 검색 주기 | 새 스킬 검색 빈도 | 매주 월요일, 격주 등 |

### 고급 옵션 (필요시 추가)

```
데이터베이스는 기본적으로 turso 를 사용하여 디자인한다.
your@email.com 등의 메일과 구글 드라이브를 활용할것.
깃허브는 your-github@email.com 으로 로그인
```
